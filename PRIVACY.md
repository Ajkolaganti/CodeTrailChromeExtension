# CodeTrail Privacy Policy

CodeTrail stores accepted coding submissions locally and syncs them to the GitHub repository selected by the user.

## Data Collected

For accepted LeetCode submissions, CodeTrail may store:

- Problem title, slug, URL, and difficulty.
- Programming language.
- The user's submitted source code.
- Runtime, memory usage, and submission timestamp when available.

CodeTrail does not collect LeetCode credentials, GitHub passwords, browsing history, cookies, analytics events, or failed submissions by default.

## Data Use

Submission data is used only to create or update files in the selected GitHub repository. CodeTrail does not send source code to an AI service.

## Local Storage

Pending sync items, settings, selected repository metadata, and GitHub authentication state are stored in Chrome local extension storage.

## Data Deletion

Use **Clear local data** in CodeTrail settings to remove local settings, pending submissions, and synced history. Use **Disconnect GitHub** to remove local authentication data. Remove synced files directly from GitHub if you want repository data deleted.

## Public Repository Warning

If the selected GitHub repository is public, synced solutions and notes are visible to anyone.
